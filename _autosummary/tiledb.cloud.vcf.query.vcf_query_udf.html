tiledb.cloud.vcf.query.vcf_query_udf¶

tiledb.cloud.vcf.query.vcf_query_udf(dataset_uri: str, *, config: Mapping[str, Any] | None = None, attrs: Sequence[str] | str | None = None, regions: Sequence[str] | str | None = None, bed_file: str | None = None, samples: Sequence[str] | str | None = None, region_partition: Tuple[int, int] | None = None, sample_partition: Tuple[int, int] | None = None, memory_budget_mb: int = 1024, af_filter: str | None = None, transform_result: Callable[[Table], Table] | None = None, log_uri: str | None = None, log_id: str = 'query', verbose: bool = False) → table¶

Run a query on a TileDB-VCF dataset.

Parameters:

  • dataset_uri – dataset URI

  • config – config dictionary, defaults to None

  • attrs – attribute names to read, defaults to None

  • regions – genomics regions to read, defaults to None

  • bed_file – URI of a BED file containing genomics regions to read, defaults to None

  • samples – sample names to read, defaults to None

  • region_partition – region partition tuple (0-based indexed, num_partitions), defaults to None

  • sample_partition – sample partition tuple (0-based indexed, num_partitions), defaults to None

  • memory_budget_mb – VCF memory budget in MiB, defaults to 1024

  • af_filter – allele frequency filter, defaults to None

  • transform_result – function to apply to the result table; by default, does not transform the result

  • log_uri – log array URI for profiling, defaults to None

  • log_id – profiler event ID, defaults to “query”

  • verbose – verbose logging, defaults to False

Returns:

Arrow table containing the query results
