tiledb.cloud.taskgraphs.registration.update¶

tiledb.cloud.taskgraphs.registration.update(graph: TaskGraphBuilder, old_name: str | None = None, *, namespace: str | None = None) → None¶

Updates the registered task graph at the given location.

Parameters:

  • graph – The new graph to replace the old value.

  • old_name – The name of the graph to rename from, if present. If graph is to be renamed, the new name of the graph must appear in graph.name.

  • namespace – The namespace, if not your own, where the graph will be updated.
