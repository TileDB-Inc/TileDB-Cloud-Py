tiledb.cloud.client.list_public_groups¶

tiledb.cloud.client.list_public_groups(namespace: str | None = None, permission: str | None = None, group_type: str | None = None, tag: str | Sequence[str] | None = None, exclude_tag: str | Sequence[str] | None = None, search: str | None = None, flat: bool = False, parent: None | str | UUID = None, page: int | None = None, per_page: int | None = None, async_req: bool = False) → object¶

List public groups owned by a user.

Parameters:

  • namespace – The namespace whose owned groups should be returned.

  • permissions – Filter arrays for the given permission.

  • group_type – If provided, return only groups of the given type.

  • tag – If provided, include groups matching the given tags.

  • exclude_tag – If provided, exclude groups matching the given tags.

  • search – A search string.

  • flat – If false (the default), return only “top-level” groups (i.e., no sub-groups within other groups).

  • parent – If provided, only show the children of the group with the given ID.

  • page – For pagination, which page to return (1-based).

  • per_page – For pagination, how many elements to return on a page.

  • async_req – Run this asynchronously; return a Future of results.
